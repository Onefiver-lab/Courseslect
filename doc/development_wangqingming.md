# 3.registrar具体实现功能
##  系统初始化与主流程

    构造函数初始化数据管理器（内存 / PostgreSQL），并创建默认的学生、教师、教学秘书及基础课程数据，初始化失败则退出系统；
    主循环exec()提供三类角色入口（学生、教师、教学秘书），支持退出系统，且包含输入合法性校验。

## 学生端功能

    学生 ID 登录验证；
    查看所有课程、选课（校验课程存在性，选课成功后保存数据）；
    查看个人课表、查看已选课程的成绩。

## 教师端功能

    教师 ID 登录验证；
    查看自己授课的所有课程；
    为授课课程的选课学生录入成绩（支持 A-F 等级或 0-100 分格式，校验成绩格式合法性，录入后保存数据）。

## 教学秘书端功能

    秘书 ID 登录验证；
    创建课程（校验课程 ID 唯一性，限定学分范围 1-10）；
    为课程分配教师（校验课程、教师 ID 存在性）；
    查看所有课程。

##  辅助功能

    工具函数readInt实现安全的整数读取，处理非数字输入异常；
    支持显示所有课程、所有教师信息；
    所有数据操作（新增课程、选课、录成绩等）均关联数据管理器，支持数据持久化保存，并处理操作异常（如保存失败提示）。
